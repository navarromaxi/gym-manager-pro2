# Gym Manager Pro 2.0

Sistema SaaS para la gestión de gimnasios, desarrollado como proyecto propio.

## Tecnologías utilizadas

- Next.js
- TypeScript
- PostgreSQL
- Vercel

## Funcionalidades principales

- Gestión de socios
- Registro de pagos
- Control de vencimientos
- Gestión de planes
- Actividades
- Prospectos
- Reportes de gestión

## Objetivo del proyecto

Centralizar la administración de gimnasios en una plataforma web moderna, permitiendo mejorar el seguimiento de socios, pagos y procesos internos.
